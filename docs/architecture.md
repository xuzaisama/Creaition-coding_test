# 架构草案

当前版本聚焦第一小时交付范围：

- FastAPI 作为 API 入口
- SQLAlchemy 负责 SQLite 持久化
- Repository 层封装数据库访问
- Service 层承载任务业务逻辑
- Router 层提供任务 CRUD 接口

第二小时已经在此基础上补充：

- 筛选、排序、分页增强
- 任务依赖关系
- 依赖完成校验
- 依赖树查询

第三小时继续补充：

- 本地 TTL 缓存，用于任务详情、列表和依赖树查询
- 写操作后的缓存失效策略
- 统一错误响应格式与全局异常处理

第四小时补充：

- 标题与描述的关键词搜索接口
- Dockerfile 与 Docker Compose 交付配置
- 更完整的 README 和接口示例
